HaloZero




  • Start Date: February 2022
  • Last updated: July 2022
  • Team: Just me
  • Genre(s): Arcade, Side scroller, 2D, Shooter
  • Engine: School C++ framework (SDL, OpenGL)
  • Platform: PC


This game was made in a C++ framework provided by my school, it only has basic functionality like a game loop, functions to draw on screen and basic math functions.

It is a remake of a game called HaloZero. I did make some noticeable changes though:

  • Removed aim snapping and made it smooth
  • Added level editor
  • Added checkpoints
  • Added option to save your progress and continue from where you saved

Overall, I learned a lot of basic C++ game development from this first big game I made in C++.
